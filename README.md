# Spanish Car Plate

The fastest Spanish Car Plate validation out there.

## Installation

```js
// npm
npm i spanish-car-plate

// yarn
yarn add spanish-car-plate
```

## Usage

### isValid

```js
import { isValid } from "spanish-car-plate";

isValid("1234BCD"); //=> true
```
